## 2. ●●● (점 3개) 사용 규칙

### 브랜드 자산 정의
`●●●`는 DOTTING의 핵심 브랜드 자산입니다.  
"모든 이야기는 계속된다"의 시각적 표현이며, **절제해서 사용해야 가치가 유지됩니다.**

### 허용되는 사용처 (3곳만)

| 사용처 | 예시 | 비고 |
|--------|------|------|
| **1. 슬로건/인용** | "모든 이야기는 계속된다 ●●●" | 브랜드 메시지 전달 |
| **2. 로딩 상태** | "책을 정리하고 있어요 ●●●" | 애니메이션 가능 |
| **3. 후기/인용 끝** | "정말 감동받았어요 ●●●" | 감정 여운 표현 |

### 금지되는 사용처
- ❌ 일반 말줄임표 대체 (→ `...` 사용)
- ❌ 구분선/장식 패턴
- ❌ 버튼 라벨
- ❌ 에러 메시지

### 시각적 규칙
```
크기: 본문과 동일 (상속)
간격: 문자 간격 0.2em (letter-spacing: 0.2em)
색상: 주변 텍스트와 동일 (상속)
애니메이션 (로딩만): 순차 페이드 (0.3s 딜레이)
```

### CSS 예시
```css
.brand-dots {
  letter-spacing: 0.2em;
}

.brand-dots-loading {
  letter-spacing: 0.2em;
}
.brand-dots-loading span:nth-child(1) { animation: fade 1s 0s infinite; }
.brand-dots-loading span:nth-child(2) { animation: fade 1s 0.3s infinite; }
.brand-dots-loading span:nth-child(3) { animation: fade 1s 0.6s infinite; }

@keyframes fade {
  0%, 100% { opacity: 0.3; }
  50% { opacity: 1; }
}
```

## 3. 랜딩 카피 가드레일

### "무료" 표현 규칙
| ✅ 허용 | ❌ 금지 |
|---------|---------|
| "프로젝트 생성은 무료" | "무료로 시작" |
| "무료로 준비만" | "지금 시작은 무료" |
| | "무료 체험" |

**원칙:** "무료"는 **프로젝트 생성**에만 한정. 인터뷰/생성은 결제 후.

### 결제 시점 표현
| ✅ 권장 | ❌ 금지 |
|---------|---------|
| "결제 후 링크를 보내면 시작" | "책 완성 시 결제" |
| "결제 후 인터뷰 시작" | "인쇄 시 결제" |
| | "나중에 결제" |

**원칙:** PRD v3.2 선결제 모델과 정합.

### 스펙/약속 표현
모든 수치 스펙에는 면책 문구 필수:
- `*샘플 기준이며, 실제 분량은 대화에 따라 달라집니다`
- `*예시 이미지입니다`

### 금지 표현 (brand-story-ab-test.md)
- ❌ "늦기 전에", "마지막"
- ❌ "점을 찍다" (경쟁사 연상)
- ❌ "AI가 해줍니다" (전면 노출 금지)

### 권장 표현
- ✅ "지금 시작", "첫 번째 이야기"
- ✅ "계속된다 ●●●"
- ✅ "저희가 만들어드립니다" (대행 톤)

## 4. 표지 옵션 UX 스펙

### 기본값
- **타이포 표지** (기본)
  - 제목 + 부제 + ●●● 심볼
  - 운영 리스크 최소

### 옵션
- **사진 표지** (선택)
  - 자녀가 업로드
  - 응답자는 업로드 불가 (권한/프라이버시)

### 사진 업로드 품질 가이드
| 항목 | 최소 요구 | 권장 |
|------|----------|------|
| 해상도 | 1200x1600px (300dpi 기준) | 2400x3200px |
| 비율 | 3:4 | 3:4 |
| 파일 크기 | 1MB 이하 | 5MB 이하 |
| 포맷 | JPG, PNG | JPG |

### 검증 시점
- **업로드 단계:** 비율/해상도 사전 체크
- **PDF 확정 단계:** 최종 품질 경고

### 책임 범위
- 사진 품질 문제는 **업로드 시 경고** 후 사용자 책임
- 인쇄 품질은 **권장 해상도 미만 시** 사전 고지
- "일러스트 변환"은 v2에서 검토 (감정 클레임 리스크)
