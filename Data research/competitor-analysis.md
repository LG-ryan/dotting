## 1. StoryWorth

### 기본 정보
| 항목 | 내용 |
|------|------|
| **설립연도** | 2012년 |
| **본사 위치** | 미국 캘리포니아주 샌프란시스코 |
| **CEO** | Nick Baum (창업자) |
| **직원 수** | 약 20~50명 (추정) |
| **펀딩** | 비공개 (부트스트랩 or 소규모 시드) |
| **매출** | 비공개 (연간 수백만 달러 추정) |

### 서비스 상세
| 항목 | 내용 |
|------|------|
| **핵심 방식** | 이메일 기반 주간 질문 → 텍스트 답변 → 1년 후 책 제작 |
| **질문 수** | 52개 (1년간 매주 1개) |
| **결과물** | 하드커버 책 1권 (기본 포함), 추가 구매 가능 |
| **기술 수준** | Low-tech (이메일 기반, AI 없음) |

### 가격 정책
| 플랜 | 가격 | 포함 내용 |
|------|------|----------|
| **연간 구독** | $99/년 | 52주 질문 + 하드커버 책 1권 |
| **추가 책** | $39/권 | 흑백 인쇄 |
| **컬러 책** | $79/권 | 컬러 인쇄 |
| **추가 사진** | 무료 | 답변에 사진 첨부 가능 |

### 창업 스토리
- Nick Baum이 할머니의 이야기를 기록하고 싶어서 시작
- "선물"로 포지셔닝하여 Mother's Day, Father's Day 시즌에 매출 집중
- 입소문 마케팅으로 성장, 대규모 광고 없이 10년 이상 운영

### 강점
- ✅ 간단한 이메일 기반 (앱 설치 불필요)
- ✅ 노인 친화적 인터페이스
- ✅ "선물" 포지셔닝 명확
- ✅ 10년 이상의 브랜드 신뢰도
- ✅ 물리적 책이라는 실체 있는 결과물

### 약점
- ❌ 1년이라는 긴 대기 시간
- ❌ AI 없음 - 고정된 질문 목록에서 선택만 가능
- ❌ 텍스트 기반 - 음성/영상 지원 없음
- ❌ 노인이 직접 타이핑해야 하는 부담
- ❌ 개인화 부족 - 맥락 없는 일반적 질문
- ❌ 가족 상호작용 기능 미흡

## 2. Remento

### 기본 정보
| 항목 | 내용 |
|------|------|
| **설립연도** | 2021년 |
| **본사 위치** | 미국 캘리포니아주 로스앤젤레스 |
| **CEO** | 비공개 (스타트업 초기) |
| **직원 수** | 약 10~20명 (추정) |
| **펀딩** | 시드 라운드 완료 (금액 비공개) |
| **매출** | 비공개 |

### 서비스 상세
| 항목 | 내용 |
|------|------|
| **핵심 방식** | 음성 녹음 → AI Speech-to-Story™ → 책/디지털 |
| **핵심 기술** | Speech-to-Story™ (음성→텍스트→스토리 자동 변환) |
| **결과물** | 하드커버 책 + 디지털 스토리북 + 영상 클립 |
| **기술 수준** | Medium-tech (AI 음성 변환) |

### 가격 정책
| 플랜 | 가격 | 포함 내용 |
|------|------|----------|
| **Basic** | $149 | 디지털 스토리북 + 소프트커버 책 1권 |
| **Premium** | $199 | 디지털 스토리북 + 하드커버 책 1권 + 영상 클립 |
| **Gift Bundle** | $249 | Premium + 추가 책 1권 + 선물 포장 |

### 핵심 기술: Speech-to-Story™
1. 사용자가 음성으로 이야기 녹음
2. AI가 음성을 텍스트로 변환 (Whisper 추정)
3. AI가 텍스트를 정제하고 문법 교정
4. 자동으로 챕터/섹션 구조화
5. 사용자 편집 가능
6. 최종 책/디지털 결과물 생성

### 강점
- ✅ 음성 기반 - 타이핑 부담 제거
- ✅ AI 활용한 자동 스토리 생성
- ✅ 멀티미디어 지원 (사진, 영상)
- ✅ 빠른 결과물 (1년 기다릴 필요 없음)
- ✅ 앱 없이 웹 링크로 접근 가능

### 약점
- ❌ AI 생성 텍스트의 부자연스러움
- ❌ 높은 가격 ($149~$199)
- ❌ 인터뷰 질문의 개인화 부족
- ❌ 후속 질문 없음 (일방향)
- ❌ 가족 협업 기능 제한적

## 6. CES 시니어 기술 동향 (에이지테크)

### 2024~2025 CES 에이지테크 주요 트렌드

#### 1. AI 돌봄 로봇
| 제품/기업 | 특징 |
|----------|------|
| **효돌 (한국)** | 정서적 교감 AI 인형, 말벗 기능 |
| **삼성 Ballie** | AI 반려로봇, 건강 모니터링 |
| **LG CLOi** | 시니어 케어 로봇, 이동 보조 |
| **ElliQ (Intuition Robotics)** | 시니어 전용 AI 컴패니언 |

#### 2. 디지털 헬스케어
- **원격 건강 모니터링**: 웨어러블 기기로 실시간 건강 추적
- **낙상 감지**: AI 기반 자동 알림 시스템
- **약 복용 알림**: 스마트 약통, 음성 알림

#### 3. 소셜 연결 기술
- **화상 통화 간소화**: 원버튼 가족 연결
- **음성 기반 인터페이스**: 터치 대신 음성 명령
- **디지털 레거시**: 추억 기록 및 공유 플랫폼

#### 4. 인지 기능 지원
- **뇌 건강 게임**: 인지 훈련 앱
- **회상 치료**: 사진/음악 기반 기억 자극
- **AI 말벗**: 대화를 통한 인지 자극

### 에이지테크 시장 규모
- **2023년**: 글로벌 약 $300억
- **2030년 예상**: 약 $900억 (연평균 15% 성장)
- **한국 시장**: 2025년 초고령사회 진입으로 급성장 예상

### 우리 서비스에 적용할 기회
1. **음성 기반 인터페이스**: 타이핑 없이 음성으로 이야기
2. **AI 대화형 인터뷰**: 맥락을 이해하는 후속 질문
3. **회상 치료 연계**: 사진/음악으로 기억 자극
4. **가족 연결**: 세대 간 소통 촉진

## 7. 시사점 및 기회
