## 1. 미끼상품 전략 (Lead Magnet) 🎣

| 원칙 | 설명 |

## 2. 브랜딩 심층 분석 🎨

### 2.1 네이밍 철학

> **좋은 브랜드명 = 처음엔 궁금하고, 알면 깊어지는 것**

**PRND 분석 (벤치마크)**:
- 처음 들으면: "뭐지? 알파벳?" → 궁금함
- 알고 나면: "아! 자동차 기어 (Park-Reverse-Neutral-Drive)" → 직관적
- 더 알면: "중고차 서비스니까 기어...! 센스 있네" → 깊은 이해

### 2.2 DOTTING vs TRIPLEDOT 심층 비교

#### DOTTING (도팅)

```
┌─────────────────────────────────────────────────────────────────┐
│                         DOTTING 분석                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  📝 의미 레이어                                                  │
│  ─────────────────────────────────                              │
│  Layer 1 (표면): "점을 찍는 중" → "무언가 진행 중"              │
│  Layer 2 (연상): "..." 로딩 → 기다리는 중 → 이야기가 만들어지는 중│
│  Layer 3 (깊이): 점 하나하나가 추억, 연결되어 이야기가 됨        │
│                                                                 │
│  ✅ 장점                                                        │
│  • "-ING" 진행형 → 동적, 살아있는 느낌                          │
│  • 짧고 기억하기 쉬움 (2음절)                                   │
│  • 로고 애니메이션 가능 (●→●●→●●●)                              │
│  • 서브 브랜드 확장 용이 (DOTTING Book, Story...)               │
│  • 도메인 확보 가능성 높음                                      │
│                                                                 │
│  ⚠️ 단점                                                        │
│  • 처음 들었을 때 "추억/이야기" 연결 약함                       │
│  • "도팅"이 한국어로 특별한 의미 없음                           │
│  • 영어권에서 "dotting the i's" (꼼꼼히 마무리) 연상 가능       │
│                                                                 │
│  🎯 첫인상 연상 테스트                                          │
│  "DOTTING이라는 서비스야" →                                     │
│  → "점 관련?" (60%)                                             │
│  → "로딩?" (20%)                                                │
│  → "모르겠음" (20%)                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### TRIPLEDOT (트리플닷)

```
┌─────────────────────────────────────────────────────────────────┐
│                       TRIPLEDOT 분석                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  📝 의미 레이어                                                  │
│  ─────────────────────────────────                              │
│  Layer 1 (표면): "세 개의 점" → "..." 말줄임표                  │
│  Layer 2 (연상): 말줄임표 → "이야기가 계속된다"                 │
│  Layer 3 (깊이): 미완의 이야기 → 당신이 완성하는 이야기         │
│                                                                 │
│  ✅ 장점                                                        │
│  • "..." 연결 직관적 (누구나 아는 기호)                         │
│  • "이야기가 계속된다" 메시지 명확                              │
│  • 로고 심플 (● ● ●)                                            │
│  • 감성적 느낌 강함                                              │
│                                                                 │
│  ⚠️ 단점                                                        │
│  • 3음절로 DOTTING보다 김                                       │
│  • "트리플닷" 한국어 발음 약간 어색                             │
│  • 서브 브랜드 확장 시 길어짐 (TRIPLEDOT Book...)               │
│  • 도메인 경쟁 가능성 (triple dot)                              │
│                                                                 │
│  🎯 첫인상 연상 테스트                                          │
│  "TRIPLEDOT이라는 서비스야" →                                   │
│  → "점 세 개? ..." (70%)                                        │
│  → "말줄임표?" (15%)                                            │
│  → "모르겠음" (15%)                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
