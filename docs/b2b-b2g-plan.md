# DOTTING B2B/B2G 사업계획서

> **버전**: 1.0  
> **작성일**: 2026년 1월 10일  
> **상태**: 장기 계획 (MVP 이후)

---

## ⚠️ 중요 공지

**이 문서는 MVP 이후를 위한 장기 계획입니다.**

B2B/B2G는 다음 이유로 MVP 범위에서 제외되었습니다:

1. **영업력 필요**: 1인 조직이 초기에 B2B 영업을 동시에 진행하기 어려움
2. **PMF 우선**: B2C에서 제품-시장 적합성 검증이 선행되어야 함
3. **복잡도 증가**: 기업용 기능 (어드민, 데이터 격리 등)은 개발 복잡도를 크게 높임
4. **정부 프로세스**: B2G는 입찰/계약 프로세스가 길고 복잡함

**진입 시점**: B2C 월 매출 500만원+ 달성 후 검토

```

│   Phase 1: B2C 집중 (현재 ~ Y1)                                         │
```

## 5. 리스크 요인
