# PART 0: 고객의 진짜 문제 (Job-to-be-Done)

## 0.1 고객이 해결하고 싶은 것

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   "물어보고 싶은데, 어떻게 물어봐야 할지 모르겠다"                       │
│                                                                         │
│   이것이 고객의 진짜 문제입니다.                                        │
│                                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   상황 (When)                                                           │
│   ─────────────────────────────────────────────                         │
│   • 부모님이 연로해지셨을 때                                            │
│   • 특별한 이벤트 (정년퇴직, 환갑, 칠순) 가 다가올 때                   │
│   • "우리 부모님 이야기를 기록해두고 싶다"는 생각이 들 때               │
│   • 나 자신의 지난 시간을 정리하고 싶을 때                              │
│   • 반려동물과의 시간을 남기고 싶을 때                                  │
│                                                                         │
│   동기 (I want to)                                                      │
│   ─────────────────────────────────────────────                         │

│   • 나도 몰랐던 이야기를 알고 싶다                                      │
│   • 세상에 하나뿐인 의미 있는 선물을 드리고 싶다                        │
│   • 특별한 날에 감동을 드리고 싶다                                      │
│                                                                         │
│   기대 결과 (So that)                                                   │
│   ─────────────────────────────────────────────                         │
│   • 살아계실 때 이야기를 남길 수 있다                                   │
│   • 가족 모두가 소중히 간직할 수 있는 유산이 된다                       │
│   • 이야기의 주인공도 자신의 인생을 돌아보며 뿌듯해한다                 │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

# PART 1: 경쟁사 브랜드 스토리 분석

## 1.1 StoryWorth: "가장 의미 있는 선물"

### 브랜드 메시지 분석

**메인 태그라인**:  
> "The most meaningful gift you can give"  
> (당신이 줄 수 있는 가장 의미 있는 선물)

**핵심 스토리라인**:
StoryWorth는 철저하게 **"선물"** 프레이밍을 사용합니다. 서비스 그 자체보다 "이것을 선물하면 받는 사람이 얼마나 감동할까"에 초점을 맞춥니다.

**그들의 스토리 전개**:
1. **문제 제기**: "당신은 부모님의 젊은 시절을 얼마나 알고 있나요?"
2. **감정 자극**: "알고 싶었지만 어떻게 물어봐야 할지 몰랐던 이야기들"
3. **해결책**: "StoryWorth가 매주 질문을 보내드립니다"
4. **결과물**: "1년 후, 세상에 하나뿐인 책이 완성됩니다"
5. **감정 착지**: "손주 대대로 전해질 가족의 보물"

**마케팅 키워드**:
- "Uncover stories you never knew" (몰랐던 이야기를 발견하세요)
- "Before it's too late" (늦기 전에)
- "A gift they'll treasure forever" (영원히 간직할 선물)
- "Weekly questions, lasting memories" (매주의 질문, 영원한 추억)

**우리가 배울 점**:
- ✅ "선물" 프레이밍의 강력함 - 구매자는 자녀, 사용자는 부모
- ✅ 긴박감 부여 - "늦기 전에"
- ✅ 결과물 구체화 - "책"이라는 실체
- ⚠️ 하지만 너무 부모님에만 한정됨
- ⚠️ "늦기 전에"는 부정적 뉘앙스

## 1.2 Remento: "삶의 이야기를 포착하다"

### 브랜드 메시지 분석

**메인 태그라인**:  
> "Capture life stories that matter"  
> (중요한 삶의 이야기를 포착하세요)

**핵심 스토리라인**:
Remento는 **"영상"**에 초점을 맞춥니다. 텍스트만으로는 담을 수 없는 목소리, 표정, 감정을 그대로 담겠다는 것이 핵심입니다.

**그들의 스토리 전개**:
1. **가치 제안**: "목소리와 표정까지 담아 영원히"
2. **차별화**: "글로는 전달되지 않는 것들이 있습니다"
3. **기술 강조**: "AI가 질문하고, 당신은 대답만 하세요"
4. **감정 연결**: "언젠가 손주가 당신의 목소리를 듣게 됩니다"

**마케팅 키워드**:
- "Video memoir" (영상 회고록)
- "Hear their voice, see their smile" (목소리를 듣고, 미소를 보세요)
- "More than just words" (단순한 글 이상)
- "AI-powered storytelling" (AI 기반 스토리텔링)

**우리가 배울 점**:
- ✅ 기술적 차별화 명확함 (영상)
- ✅ 감각적 경험 강조 (목소리, 표정)
- ⚠️ 진입 장벽이 높음 (카메라 앞에 서야 함)
- ⚠️ 시니어에게 부담스러울 수 있음

## 1.3 MemLife: "기억을 생명으로"

### 브랜드 메시지 분석

**메인 태그라인**:  
> "Memories become life"  
> (기억이 생명이 됩니다)

**핵심 스토리라인**:
MemLife는 **"치료적 가치"**를 강조합니다. 추억을 기록하는 것이 단순한 보존이 아니라, 정서적 치유와 삶의 의미 발견이라는 메시지입니다.

**그들의 스토리 전개**:
1. **철학적 접근**: "기억은 우리가 누구인지를 정의합니다"
2. **치료적 가치**: "이야기를 하면서 자신을 다시 발견합니다"
3. **관계 회복**: "말하지 못했던 것들을 꺼내놓게 됩니다"
4. **생명력 부여**: "당신의 이야기는 영원히 살아있게 됩니다"

**마케팅 키워드**:
- "Memory is identity" (기억은 정체성)
- "Therapeutic storytelling" (치료적 스토리텔링)
- "Legacy preservation" (유산 보존)
- "Live forever through stories" (이야기로 영원히 살다)

**우리가 배울 점**:
- ✅ 깊은 철학적 접근
- ✅ 치료적/정서적 가치 강조
- ⚠️ 너무 무겁고 진지함
- ⚠️ "죽음" 연상 가능성

# PART 2: DOTTING 브랜드 철학

## 2.1 "점"의 새로운 정의

> **⚠️ 주의**: "이야기에 점을 찍다"는 한국어로 "끝낸다"는 부정적 의미입니다.  
> **✅ 우리의 해석**: 점은 "마침표"가 아니라 **"시작점"**, **"연결점"**, **"모이는 것"**입니다.

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│     ●                    점 하나는 그냥 점                               │
│                                                                         │
│     ●   ●                두 개의 점은 연결의 시작                        │
│                                                                         │
│     ●   ●   ●            세 개의 점은 ...                               │
│                                                                         │
│     ●──●──●──●           이어지면 선이 되고                              │
│                                                                         │
│     ●──●──●              ┐                                               │
│     │  │  │              │  모이면                                       │
│     ●──●──●              │  이야기가 됩니다                              │
│     │  │  │              │                                               │
│     ●──●──●              ┘                                               │
│                                                                         │
│     ═══════════════════════════════════════════════════════════════     │
│                                                                         │
│     DOTTING = "점을 모으는 중"                                           │
│     당신의 순간들이 모여 이야기가 되는 과정                              │
│                                                                         │
│     ...                                                                 │
│     그리고 그 이야기는 계속됩니다                                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 점의 5가지 의미

| # | 의미 | 설명 | 브랜드 연결 |
|---|------|------|------------|
| 1 | **시작점** | 모든 이야기는 하나의 점에서 시작 | "첫 번째 이야기를 시작하세요" |
| 2 | **연결점** | 세대와 세대, 사람과 사람을 연결 | "점이 모여 관계가 됩니다" |
| 3 | **기록점** | 매 순간이 하나의 기록 | "오늘도 한 점이 더해집니다" |
| 4 | **별처럼** | 흩어진 점들이 별자리가 되듯 | "당신만의 별자리를 그려보세요" |
| 5 | **로딩...** | 진행 중, 계속됨 | "● ● ● 당신의 이야기가 만들어지고 있어요" |

### 챕터 1: 물어보지 못한 질문

```

나는 그 이야기를 처음 들었다.
30년을 곁에 있었는데.

"할머니, 그때 얘기 좀 해주세요"
한 번도 말하지 못했다.

장례식장을 나오면서 생각했다.

'이제 영영 들을 수 없는 건가?'

할머니가 돌아가시고 나니,
아버지에게, 어머니에게 물어보고 싶은 것들이
갑자기 떠올랐다.

"아빠는 왜 그 회사에 들어갔어?"
"엄마는 어떻게 아빠 만났어?"
"나 낳고 제일 힘들었던 거 뭐야?"

물어볼 수 있는데,
```

## 8.2 해결책 1: AI 깊이 파기 (Deep Dive) 전략

### 핵심 아이디어
> **질문 수는 적게, 하지만 AI가 꼬리 질문을 해서 깊이 파고든다**

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   기존 방식 (StoryWorth 스타일)                                         │
│   ─────────────────────────────────────────────────────────────         │
│                                                                         │
│   Q1: 첫 월급 받았을 때 뭐 샀어요?                                       │
│   A1: "구두 샀어요."                                                    │
│   → 끝. 다음 질문으로.                                                  │
│                                                                         │
│   결과: 짧은 답변 52개 = 얇은 책                                        │
│                                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   DOTTING 방식 (AI Deep Dive)                                           │
│   ─────────────────────────────────────────────────────────────         │
│                                                                         │
│   Q1: 첫 월급 받았을 때 뭐 샀어요?                                       │
│   A1: "구두 샀어요."                                                    │
│                                                                         │
│   AI: "구두요! 그때 구두가 비쌌을 것 같은데,                             │
│        어떤 구두였어요? 어디서 샀어요?"                                  │
│   A2: "남대문시장에서 샀어. 검은색 가죽구두.                             │
│        그때 진짜 비쌌어. 월급 반이었어."                                 │
│                                                                         │
│   AI: "월급 반이요? 그러면 월급이 얼마였어요?                            │
│        지금 생각하면 어떠세요?"                                          │
│   A3: "그때 월급이 8만원이었으니까...                                    │
│        지금으로 치면 첫 월급으로 400만원짜리                             │
│        명품 구두 산 셈이지. 근데 그때는 다들 그랬어.                     │
│        구두가 신분의 상징이었거든..."                                    │
│                                                                         │
│   AI: "신분의 상징이요? 그 구두 신고 처음 출근했을 때                    │
│        기분이 어땠어요?"                                                │
│   A4: "그때 진짜 기분이... (감정 회상)                                   │
│        뭔가 내가 어른이 된 것 같았어.                                    │
│        아버지한테 보여드리고 싶었는데..."                                │
│                                                                         │
│   → 하나의 질문에서 4-5개의 깊은 이야기 추출                            │
│                                                                         │
│   결과: 질문 12개 × 깊이 5배 = 풍성한 책                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 기술 구현

| 기능 | 구현 방식 | 효과 |
|------|----------|------|
| **Follow-up 질문 생성** | GPT-4로 답변 분석 → 후속 질문 자동 생성 | 자연스러운 대화 흐름 |
| **감정 포착** | 답변에서 감정 키워드 감지 → 감정 탐구 질문 | 깊이 있는 이야기 |
| **맥락 연결** | 이전 답변 기억 → 연관 질문 | 이야기 일관성 |
| **적정 깊이 판단** | 답변 길이/감정 강도로 더 파거나 넘어감 | 지치지 않게 |

### 대화 예시 프롬프트 (AI 인터뷰어)

```
당신은 따뜻하고 공감 능력 있는 인터뷰어입니다.

목표: 사용자의 인생 이야기를 자연스러운 대화로 끌어내기

원칙:
1. 짧은 답변이 나오면 구체적으로 더 물어보세요
   - "그때 어디서요?", "어떤 기분이었어요?", "그래서 어떻게 됐어요?"
   
2. 감정 단어가 나오면 감정을 탐구하세요
   - "힘들었다" → "어떤 점이 가장 힘드셨어요?"
   - "기뻤다" → "그 기쁨을 누구랑 나누셨어요?"
   
3. 사람 이름이 나오면 관계를 물어보세요
   - "그 분은 어떤 분이셨어요?"
   
4. 숫자가 나오면 맥락을 물어보세요
   - "그때 얼마였어요?" → "지금으로 치면 어느 정도일까요?"
   
5. 3-4번 깊이 파고들면 자연스럽게 다음 주제로 넘어가세요
   - "좋은 이야기 감사해요. 그럼 이번엔..."
```

```python
# 의사 코드
def get_next_question(user_id):
    history = get_user_history(user_id)
    
    # 평균 답변 길이 분석
    avg_response_length = calculate_avg_length(history)
    
    # 세션 지속 시간 분석
    avg_session_duration = calculate_avg_duration(history)
    
    # 감정 상태 분석
    recent_emotions = analyze_emotions(history[-3:])
    
    if avg_response_length > 200:
        # 말 많은 사용자 → 열린 질문
        question_type = "open_ended"
    else:
        # 말 적은 사용자 → 구체적 질문
        question_type = "specific"
    
    if avg_session_duration < 120:  # 2분 미만
        # 빨리 끝내는 경향 → 가벼운 질문
        topic_weight = "light"
    else:
        # 오래 하는 경향 → 깊은 질문 가능
        topic_weight = "deep"
    
    return generate_question(question_type, topic_weight)
```
